## 功能

1. 合并，可将svg文件批量拼合为字体文件
2. 修改，可对字体追加、删除、替换、导出字形，例如：替换字体中的指定字形
3. 裁剪，从中文字体中过滤出需要的字形（过滤8.1M的中文字体，需等待时间约6s）
